# Custom-MLP4DSM-Firmware
Custom firmware for the Spektrum MLP4DSM which enables flying a "Blade nano CP X" 

6-Channel 4-Model Memory

Three Modell Types Clasic, Airplane, Helicoper

DSM2, DSMX and France RF Mode

Configurable stick input mode 1-4 

Range Check 

Two flight modes: NORM / STUNT

Programmable EXPO and Rate for both flight modes

Count Down Timer

Programmable  input channel reversing

Batery monitor


Helicopter:

Programmable 3-Point THR and PITCH curves for both flight modes

Throttle Cut


Airplane:

Three 3-Channel mixers
